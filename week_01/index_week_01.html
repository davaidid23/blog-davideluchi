Esercizio 1.1: Modifica testo automatica

Questo testo è ininfluente e verrà sostituito da un div esterno
HTML (Struttura 1.1) 
<div id="target">Questo testo è ininfluente e verrà sostituito da un div esterno</div>
funzioni.js (Logica 1.1) 
const elemento = document.getElementById('target');
if (elemento) {
    elemento.textContent = "Questo testo non è stato scritto sul file .html, ma è stato inserito tramite un file JS esterno!";
    elemento.style.color = "#ff4500";
}

Testo da modificare

HTML (Struttura 1.2) 
<p id="testo-interattivo">Testo da modificare</p>
<button id="btn-testo">Cambia Testo</button>
<button id="btn-colore">Cambia Colore</button>
funzioni.js (Logica 1.2) 
const btnT = document.getElementById('btn-testo');
const btnC = document.getElementById('btn-colore');

btnT.addEventListener('click', function() {
    document.getElementById('testo-interattivo').textContent = "Questo testo non è stato scritto sul file .html, ma è stato inserito tramite un file JS esterno!";
});

btnC.addEventListener('click', function() {
    const colori = ["#27ae60", "#8e44ad", "#c0392b"];
    document.getElementById('testo-interattivo').style.color = colori[0]; // Esempio
});

Esercizio 2: Dimostrazione della varianza di Welford

L'algoritmo di Welford permette di calcolare la varianza in modo ricorsivo. Questo è fondamentale quando si ha a che fare con dataset di grandi dimensioni, poiché calcolare la varianza in modo standard potrebbe portare a errori di precisione numerica (cancellazione catastrofica). Di seguito è riportata la dimostrazione matematica rigorosa.

1. Qualche definizione basilare

Sia $x_1, x_2, \dots, x_N$ la sequenza di dati composta da $N$ campioni. Definiamo la media campionaria al passo $n < N$ come:

$$\bar{x}_n = \frac{1}{n}\sum_{i=1}^n x_i$$

Definiamo la somma dei quadrati delle differenze dalla media al passo $n$ come:

$$M_n = \sum_{i=1}^n (x_i - \bar{x}_n)^2$$

2. Relazione tra $\bar{x}_n$ e $\bar{x}_{n-1}$

Dobbiamo trovare il legame tra $\bar{x}_n$ e $\bar{x}_{n-1}$ per costruire la relazione ricorsiva. Possiamo osservare che isolando la somma dei dati, possiamo separare l'ennesimo termine $x_n$ per trovare il legame tra $\bar{x}_n$ e $\bar{x}_{n-1}$:

$$\bar{x}_n = \frac{(n-1)\bar{x}_{n-1} + x_n}{n}$$ $$n\bar{x}_n = (n-1)\bar{x}_{n-1} + x_n$$

A questo punto sottraendo $\bar{x}_n$ da entrambi i membri e portando $(n-1)\bar{x}_{n-1}$ dall'altra parte, si ottiene:

$$(n-1)(\bar{x}_n - \bar{x}_{n-1}) = x_n - \bar{x}_n$$

3. Scomposizione di $M_n$

Tolgo l'$n-1$-esimo elemento dalla sommatoria:

$$M_n = \sum_{i=1}^{n-1} (x_i - \bar{x}_n)^2 + (x_n - \bar{x}_n)^2$$

Aggiungo e tolgo $\bar{x}_{n-1}$ all'interno della sommatoria:

$$M_n = \sum_{i=1}^{n-1} [(x_i - \bar{x}_{n-1}) - (\bar{x}_n - \bar{x}_{n-1})]^2 + (x_n - \bar{x}_n)^2$$

Sviluppando il quadrato del binomio otteniamo:

$$M_n = \sum_{i=1}^{n-1} (x_i - \bar{x}_{n-1})^2 - 2(\bar{x}_n - \bar{x}_{n-1})\sum_{i=1}^{n-1} (x_i - \bar{x}_{n-1}) + \sum_{i=1}^{n-1} (\bar{x}_n - \bar{x}_{n-1})^2 + (x_n - \bar{x}_n)^2$$

4. Semplificazione

Ora si possono osservare i singoli pezzi dell'equazione precedente:

  • Il primo termine è proprio $M_{n-1}$
  • Il secondo termine è nullo poiché la somma degli scarti dalla media $\sum (x_i - \bar{x}_{n-1})$ è per definizione zero
  • Il terzo termine, essendo una costante sommata $n-1$ volte, diventa $(n-1)(\bar{x}_n - \bar{x}_{n-1})^2$

L'equazione diventa quindi:

$$M_n = M_{n-1} + (n-1)(\bar{x}_n - \bar{x}_{n-1})^2 + (x_n - \bar{x}_n)^2$$

5. Conclusione

Sostituendo la relazione ricavata nella sezione 2 e raccogliendo i termini si ottiene l'algoritmo per la varianza di Welford:

$$M_n = M_{n-1} + (x_n - \bar{x}_n)(x_n - \bar{x}_{n-1})$$

La varianza campionaria $s^2$ si può ottenere dividendo $M_n$ per i gradi di libertà:

$$s^2 = \frac{M_n}{n-1}$$
