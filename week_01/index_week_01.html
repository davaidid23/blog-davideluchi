Esercizio 1: modificare un testo cambiando parole e colore utilizzando un file JS esterno.

Nella casella sottostante appare il risultato del programmino creato. Il codice scritto è riportato successivamente.

Questo testo originale verrà rimpiazzato dal JavaScript...
index.html 
<!DOCTYPE html>
<html lang="it">
<head>
    <meta charset="UTF-8">
    <title>Esercizio 1: modificare un testo cambiando parole e colore utilizzando un file JS esterno.</title>
</head>
<body>

    <div id="target">Questo testo è ininfluente: se tutto va come deve, verrà sostituito dal testo scritto nel file JS esterno </div>

    <script src="funzioni.js"></script>
    
</body>
</html>
funzioni.js 
// Seleziono l'elemento tramite il suo ID, che in questo caso ho chiamato 'target'
const elemento = document.getElementById('target');

// Cambio il testo
elemento.textContent = "Questo testo non è stato scritto sul file .html, ma è stato inserito tramite un file JS esterno!";

// Imposto colore e stile per il nuovo testo cambiato
elemento.style.color = "#ff4500";
elemento.style.fontSize = "24px";
elemento.style.fontWeight = "bold";

Dimostrazione della Varianza di Welford

L'algoritmo di Welford è fondamentale per calcolare la varianza campionaria in un singolo passaggio (single-pass), prevenendo gli errori di approssimazione causati dalla cancellazione catastrofica dei numeri floating-point. Di seguito la dimostrazione matematica rigorosa del calcolo iterativo della somma dei quadrati.

1. Definizioni Preliminari

Sia $x_1, x_2, \dots, x_n$ una sequenza di dati. Definiamo la media campionaria al passo $n$ come:

$$\bar{x}_n = \frac{1}{n}\sum_{i=1}^n x_i$$

Definiamo la somma dei quadrati delle differenze dalla media al passo $n$ (che indicheremo con $M_n$) come:

$$M_n = \sum_{i=1}^n (x_i - \bar{x}_n)^2$$

Il nostro obiettivo è trovare una relazione ricorsiva che leghi $M_n$ a $M_{n-1}$.

2. Relazione sulla Media

Isolando la somma dei dati, possiamo separare l'ennesimo termine:

$$n\bar{x}_n = \sum_{i=1}^{n-1} x_i + x_n = (n-1)\bar{x}_{n-1} + x_n$$

Da questa equazione, manipolando algebricamente i termini per trovare la differenza tra la media attuale e la precedente, otteniamo una relazione cruciale per i passaggi successivi:

$$(n-1)(\bar{x}_n - \bar{x}_{n-1}) = x_n - \bar{x}_n$$

3. Scomposizione di $M_n$

Espandiamo la formula di $M_n$ separando l'ultimo termine dalla sommatoria e inserendo $\bar{x}_{n-1}$ con un trucco algebrico (sommando e sottraendo all'interno del quadrato):

$$M_n = \sum_{i=1}^{n-1} [(x_i - \bar{x}_{n-1}) - (\bar{x}_n - \bar{x}_{n-1})]^2 + (x_n - \bar{x}_n)^2$$

Sviluppando il quadrato del binomio all'interno della sommatoria otteniamo:

$$M_n = \sum_{i=1}^{n-1} (x_i - \bar{x}_{n-1})^2 - 2(\bar{x}_n - \bar{x}_{n-1})\sum_{i=1}^{n-1} (x_i - \bar{x}_{n-1}) + \sum_{i=1}^{n-1} (\bar{x}_n - \bar{x}_{n-1})^2 + (x_n - \bar{x}_n)^2$$

4. Semplificazione

Analizziamo i termini della sommatoria espansa:

  • Il primo termine è esattamente la definizione di $M_{n-1}$.
  • Il secondo termine è nullo, poiché la somma degli scarti di una variabile dalla propria media è sempre zero: $\sum_{i=1}^{n-1} (x_i - \bar{x}_{n-1}) = 0$.
  • Il terzo termine è una costante sommata $n-1$ volte, pari a $(n-1)(\bar{x}_n - \bar{x}_{n-1})^2$.

L'equazione si riduce quindi a:

$$M_n = M_{n-1} + (n-1)(\bar{x}_n - \bar{x}_{n-1})^2 + (x_n - \bar{x}_n)^2$$

5. Passaggio Finale

Utilizziamo l'identità trovata al Punto 2, sostituendo un fattore $(n-1)(\bar{x}_n - \bar{x}_{n-1})$ nel secondo termine con $(x_n - \bar{x}_n)$:

$$M_n = M_{n-1} + (x_n - \bar{x}_n)(\bar{x}_n - \bar{x}_{n-1}) + (x_n - \bar{x}_n)^2$$

Infine, raccogliamo a fattor comune $(x_n - \bar{x}_n)$ per gli ultimi due termini:

$$M_n = M_{n-1} + (x_n - \bar{x}_n)[(\bar{x}_n - \bar{x}_{n-1}) + (x_n - \bar{x}_n)]$$ $$M_n = M_{n-1} + (x_n - \bar{x}_n)(x_n - \bar{x}_{n-1})$$

C.V.D.

Nota: la varianza campionaria (per $n > 1$) si calcola dividendo il risultato ottenuto per i gradi di libertà:

$$s^2 = \frac{M_n}{n-1}$$
