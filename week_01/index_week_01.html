Esercizio 1: modificare un testo cambiando parole e colore utilizzando un file JS esterno.

Nella casella sottostante appare il risultato del programmino creato. Il codice scritto è riportato successivamente.

Questo testo originale verrà rimpiazzato dal JavaScript...
index.html 
<!DOCTYPE html>
<html lang="it">
<head>
    <meta charset="UTF-8">
    <title>Esercizio 1: modificare un testo cambiando parole e colore utilizzando un file JS esterno.</title>
</head>
<body>

    <div id="target">Questo testo è ininfluente: se tutto va come deve, verrà sostituito dal testo scritto nel file JS esterno </div>

    <script src="funzioni.js"></script>
    
</body>
</html>
funzioni.js 
// Seleziono l'elemento tramite il suo ID, che in questo caso ho chiamato 'target'
const elemento = document.getElementById('target');

// Cambio il testo
elemento.textContent = "Questo testo non è stato scritto sul file .html, ma è stato inserito tramite un file JS esterno!";

// Imposto colore e stile per il nuovo testo cambiato
elemento.style.color = "#ff4500";
elemento.style.fontSize = "24px";
elemento.style.fontWeight = "bold";

Dimostrazione della Varianza di Welford

L'algoritmo di Welford permette di calcolare la varianza in modo ricorsivo. Questo è fondamentale quando si ha a che fare con dataset di grandi dimensioni, poiché calcolare la varianza in modo standard potrebbe portare a errori di precisione numerica (cancellazione catastrofica). Di seguito è riportata la dimostrazione matematica rigorosa.

1. Qualche definizione basilare

Sia $x_1, x_2, \dots, x_N$ la sequenza di dati composta da N campioni. Definiamo la media campionaria al passo $n < N$ come:

$$\bar{x}_n = \frac{1}{n}\sum_{i=1}^n x_i$$

Definiamo la somma dei quadrati delle differenze dalla media (Sum of Squares) al passo $n$ come:

$$M_n = \sum_{i=1}^n (x_i - \bar{x}_n)^2$$

2. Relazione sulla Media

Isolando la somma dei dati, possiamo separare l'ennesimo termine $x_n$ per trovare il legame tra $\bar{x}_n$ e $\bar{x}_{n-1}$:

$$n\bar{x}_n = (n-1)\bar{x}_{n-1} + x_n$$

Sottraendo $\bar{x}_n$ da entrambi i membri e manipolando l'espressione, otteniamo la relazione fondamentale:

$$(n-1)(\bar{x}_n - \bar{x}_{n-1}) = x_n - \bar{x}_n$$

3. Scomposizione di $M_n$

Espandiamo $M_n$ inserendo il termine $\bar{x}_{n-1}$ all'interno della sommatoria e separando l'ultimo termine della serie:

$$M_n = \sum_{i=1}^{n-1} [(x_i - \bar{x}_{n-1}) - (\bar{x}_n - \bar{x}_{n-1})]^2 + (x_n - \bar{x}_n)^2$$

Sviluppando il quadrato del binomio otteniamo:

$$M_n = \sum_{i=1}^{n-1} (x_i - \bar{x}_{n-1})^2 - 2(\bar{x}_n - \bar{x}_{n-1})\sum_{i=1}^{n-1} (x_i - \bar{x}_{n-1}) + \sum_{i=1}^{n-1} (\bar{x}_n - \bar{x}_{n-1})^2 + (x_n - \bar{x}_n)^2$$

4. Semplificazione

Analizziamo i singoli componenti dell'equazione:

  • Il primo termine corrisponde a $M_{n-1}$
  • Il secondo termine è nullo poiché la somma degli scarti dalla media $\sum (x_i - \bar{x}_{n-1})$ è per definizione zero
  • Il terzo termine, essendo una costante sommata $n-1$ volte, diventa $(n-1)(\bar{x}_n - \bar{x}_{n-1})^2$

L'equazione si semplifica quindi in:

$$M_n = M_{n-1} + (n-1)(\bar{x}_n - \bar{x}_{n-1})^2 + (x_n - \bar{x}_n)^2$$

5. Conclusione

Sostituendo la relazione trovata al punto 2 e raccogliendo i termini, otteniamo la formula di aggiornamento ricorsiva di Welford:

$$M_n = M_{n-1} + (x_n - \bar{x}_n)(x_n - \bar{x}_{n-1})$$

La varianza campionaria $s^2$ si ottiene infine dividendo per i gradi di libertà:

$$s^2 = \frac{M_n}{n-1}$$
