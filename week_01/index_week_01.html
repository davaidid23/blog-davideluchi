Esercizio 1: Manipolazione del DOM

In questo esercizio, un file JavaScript esterno modifica il contenuto di un DIV esistente cambiandone il testo e lo stile.

Questo testo originale verrà rimpiazzato dal JavaScript...
index.html 
<div id="target">
    Questo testo originale verrà rimpiazzato...
</div>

<script src="funzioni.js"></script>
funzioni.js 
// Seleziono l'elemento tramite l'ID 'target'
const elemento = document.getElementById('target');

// Cambio il contenuto del testo
elemento.textContent = "Testo modificato con JS esterno!";

// Imposto colore e stile
elemento.style.color = "#ff4500";
elemento.style.fontSize = "24px";
elemento.style.fontWeight = "bold";

Dimostrazione della Varianza di Welford

L'algoritmo di Welford permette di calcolare la varianza in modo incrementale. Ecco le formule principali:

Aggiornamento della media:

$$\bar{x}_n = \bar{x}_{n-1} + \frac{x_n - \bar{x}_{n-1}}{n}$$

Aggiornamento della somma dei quadrati ($M_n$):

$$M_n = M_{n-1} + (x_n - \bar{x}_{n-1})(x_n - \bar{x}_n)$$

Varianza finale:

$$s^2 = \frac{M_n}{n-1}$$
