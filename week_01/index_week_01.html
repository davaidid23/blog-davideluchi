Esercizio 1: modificare un testo cambiando parole e colore utilizzando un file JS esterno.

Nella casella sottostante appare il risultato del programmino creato. Il codice scritto è riportato successivamente.

Questo testo originale verrà rimpiazzato dal JavaScript...
index.html 
<!DOCTYPE html>
<html lang="it">
<head>
    <meta charset="UTF-8">
    <title>Esercizio 1: modificare un testo cambiando parole e colore utilizzando un file JS esterno.</title>
</head>
<body>

    <div id="target">Questo testo è ininfluente: se tutto va come deve, verrà sostituito dal testo scritto nel file JS esterno </div>

    <script src="funzioni.js"></script>
    
</body>
</html>
funzioni.js 
// Seleziono l'elemento tramite il suo ID, che in questo caso ho chiamato 'target'
const elemento = document.getElementById('target');

// Cambio il testo
elemento.textContent = "Questo testo non è stato scritto sul file .html, ma è stato inserito tramite un file JS esterno!";

// Imposto colore e stile per il nuovo testo cambiato
elemento.style.color = "#ff4500";
elemento.style.fontSize = "24px";
elemento.style.fontWeight = "bold";

Dimostrazione della Varianza di Welford

L'algoritmo di Welford permette di calcolare la varianza in modo ricorsivo. Questo è fondamentale quando si ha a che fare con valori molto grandi, perchè calcolar la varianza in modo standard potrebbe portare alla saturazione della memoria e alla conseguente perdita o approssimazione dei valori. Di seguito è riportata la dimostrazione matematica rigorosa.

1. Qualche definizione basilare

Sia x_1, x_2, ..., x_n la sequenza di dati composta da n campioni. Definiamo la media campionaria al passo n come:

$$\bar{x}_n = \frac{1}{n}\sum_{i=1}^n x_i$$

Definiamo la somma dei quadrati delle differenze dalla media al passo n come:

$$M_n = \sum_{i=1}^n (x_i - \bar{x}_n)^2$$

2. Relazione sulla Media

Isolando la somma dei dati, possiamo separare l'ennesimo termine:

$$n\bar{x}_n = (n-1)\bar{x}_{n-1} + x_n$$

Otteniamo quindi la relazione:

$$(n-1)(\bar{x}_n - \bar{x}_{n-1}) = x_n - \bar{x}_n$$

3. Scomposizione di M_n

Espandiamo la formula separando l'ultimo termine:

$$M_n = \sum_{i=1}^{n-1} [(x_i - \bar{x}_{n-1}) - (\bar{x}_n - \bar{x}_{n-1})]^2 + (x_n - \bar{x}_n)^2$$

Sviluppando il quadrato:

$$M_n = \sum_{i=1}^{n-1} (x_i - \bar{x}_{n-1})^2 - 2(\bar{x}_n - \bar{x}_{n-1})\sum_{i=1}^{n-1} (x_i - \bar{x}_{n-1}) + \sum_{i=1}^{n-1} (\bar{x}_n - \bar{x}_{n-1})^2 + (x_n - \bar{x}_n)^2$$

4. Semplificazione

Analizziamo i termini:

  • Il primo termine è: $$M_{n-1}$$
  • Il secondo termine è nullo poiché la somma degli scarti è zero: $$\sum (x_i - \bar{x}_{n-1}) = 0$$
  • Il terzo termine è: $$(n-1)(\bar{x}_n - \bar{x}_{n-1})^2$$

L'equazione diventa:

$$M_n = M_{n-1} + (n-1)(\bar{x}_n - \bar{x}_{n-1})^2 + (x_n - \bar{x}_n)^2$$

5. Conclusione

Sostituendo i termini e raccogliendo a fattor comune, otteniamo la formula di aggiornamento finale:

$$M_n = M_{n-1} + (x_n - \bar{x}_n)(x_n - \bar{x}_{n-1})$$

La varianza campionaria si ottiene infine con:

$$s^2 = \frac{M_n}{n-1}$$
